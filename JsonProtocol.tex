\section{JSON Protocol}
    Defined here is what a JSON Object should look like for sending and receiving shapes.
For circles:
\begin{minted}
[
frame=lines,
framesep=2mm,
baselinestretch=1.2,
fontsize=\footnotesize,
]
{json}

{
    "id": 1,
    "shape": "circle",
    "data": {
        "radius": "10",
        "center": {
            "x": 100,
            "y": 150
        }
    }
}
\end{minted}

For rectangles:
\begin{minted}
[
frame=lines,
framesep=2mm,
baselinestretch=1.2,
fontsize=\footnotesize,
]
{json}

{
    "id": 2,
    "shape": "line",
    "data": {
        "start": {
            "x": 10,
            "y": 50
        },
        "end": {
            "x": 150,
            "y": 100
        }
    }
}
\end{minted}
